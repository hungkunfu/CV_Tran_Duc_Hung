Mô tả

Đây là một website sử dụng mô hình MVC :

- Ngôn ngữ C#

- Front end HTML, CSS, Boostrap

- Cơ sở dữ liệu là SQL Server 2019

- Sử dụng Query để lấy dữ liệu

- Sử dụng Ajax, Jquery, dataTables, Validate, Bootbox

- Sử dụng HttpSession làm giỏ hàng, checkout sản phẩm

- Sử dụng Identity phân quyền truy cập theo ROLE

- Sử dụng ứng dụng Visual studio

Chức năng

Trang chủ

- Hiển thị danh sách thể loại, sản phẩm nổi bật

- Thêm sản phẩm vào giỏ hàng

- Đăng kí,đăng nhập

- Đăng kí có xác nhận mail mới login được

- Checkout giỏ hàng, tăng giảm số lượng, checkout order

Trang admin

- Đăng nhập vào trang quản lý với role ADMIN

- Thêm, sửa, xóa và xem thông tin

- Phân trang, tìm kiếm

Kĩ năng thu được

Qua quá trình làm việc bản thân biết cách validate dữ liệu backend và frontend, dùng Ajax, Jquery, dataTables, Kết hợp HTML, CSS, Boostrap, Template để vẽ dữ liệu. Sử dụng HttpSession làm giỏ hàng, checkout giỏ hàng và Identity để phân quyền.
